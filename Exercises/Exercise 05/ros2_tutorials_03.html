ROS 2 Humble Tutorials, Part 3

1 PDF Viewer

When the need arises for you to view a pdf document, Ubuntu has a default viewer called Evince that you can use from the terminal window as follows

$ evince <document_name>

where <document_name> is the name of the document you would like to view.

2 Launch

ROS 2 Launch files allow you to start up and configure a number of executables containing ROS 2 nodes simultaneously.

3 URDF

URDF (Unified Robot Description Format) is a file format for specifying the geometry and organization of robots in ROS.

4 RViz

RViz is a 3D visualizer for the Robot Operating System (ROS) framework.

Note

RViz was installed on your laptop with the ROS2 Humble installation and you don’t need to install it again.
