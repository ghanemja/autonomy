1 PDF Viewer
============
When the need arises for you to view a pdf document, Ubuntu has a default viewer called ``Evince`` that you can use from the terminal window as follows
::
	$ evince <document_name>
	
where ``<document_name>`` is the name of the document you would like to view.

3 URDF
======
URDF (Unified Robot Description Format) is a file format for specifying the geometry and organization of robots in ROS.

* `Building a visual robot model from scratch <https://docs.ros.org/en/humble/Tutorials/Intermediate/URDF/Building-a-Visual-Robot-Model-with-URDF-from-Scratch.html>`_ (20 minutes)
* `Building a movable robot model <https://docs.ros.org/en/humble/Tutorials/Intermediate/URDF/Building-a-Movable-Robot-Model-with-URDF.html>`_ (10 minutes)
* `Adding physical and collision properties <https://docs.ros.org/en/humble/Tutorials/Intermediate/URDF/Adding-Physical-and-Collision-Properties-to-a-URDF-Model.html>`_ (10 minutes)
* `Using Xacro to clean up your code <https://docs.ros.org/en/humble/Tutorials/Intermediate/URDF/Using-Xacro-to-Clean-Up-a-URDF-File.html>`_ (20 minutes)
* `Using URDF with robot_state_publisher <https://docs.ros.org/en/humble/Tutorials/Intermediate/URDF/Using-URDF-with-Robot-State-Publisher.html>`_ (15 minutes)
* `Generating an URDF File <https://docs.ros.org/en/humble/Tutorials/Intermediate/URDF/Exporting-an-URDF-File.html>`_ (5 minutes)
